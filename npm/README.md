# dossiers

Run the [Dossiers](https://dossie.rs) CLI without installing Rust — ideal for CI:

```sh
npx dossiers --help
```

On install, this package downloads the prebuilt binary matching your platform
from the [GitHub release](https://github.com/dossie-rs/cli/releases) for the
same version. Supported platforms: Linux and macOS (x86_64 / aarch64) and
Windows (x86_64 / aarch64).

Prefer a native install? `cargo install dossiers`.

Source and issues: https://github.com/dossie-rs/cli
